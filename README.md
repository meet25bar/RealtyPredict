# RealtyPredict 🏡

RealtyPredict is a Machine Learning project that predicts house prices using the Ames Housing dataset and Linear Regression.  
The project focuses on data preprocessing, missing value handling, feature engineering, model training, and regression evaluation.

---

## 🚀 Features

- Data Cleaning & Preprocessing
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Linear Regression Model
- Model Evaluation using:
  - R² Score
  - MAE
  - MSE
  - RMSE
- Data Visualization

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Dataset

The project uses the Ames Housing Dataset containing various property features such as:

- Lot Area
- Overall Quality
- Year Built
- Garage Area
- Basement Features
- Living Area
- Neighborhood
- and many more...

Target Variable:
- `SalePrice`

---

## 📊 Model Used

### Linear Regression

The model was trained to predict house prices based on multiple housing features.

---

## 📈 Evaluation Metrics

The model performance was evaluated using:

```python
R² Score
Mean Absolute Error (MAE)
Mean Squared Error (MSE)
Root Mean Squared Error (RMSE)
```

---

## 📷 Sample Workflow

1. Load Dataset
2. Clean Missing Values
3. Encode Categorical Features
4. Train/Test Split
5. Train Linear Regression Model
6. Generate Predictions
7. Evaluate Model Performance
